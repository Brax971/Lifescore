# LifeScore MVP

MVP de LifeScore : un questionnaire simple qui calcule un LifeScore (0–100)
et affiche un bilan structuré par domaines, sans création de compte ni stockage
côté serveur.

## Développement

```bash
npm install
npm run dev
```

Puis ouvre http://localhost:3000 dans ton navigateur.

Déployable tel quel sur Vercel avec Next.js 14 (App Router).
